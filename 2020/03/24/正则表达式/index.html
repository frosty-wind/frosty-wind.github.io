正则表达式

发表于 2020-03-24 | 分类于 基础知识

正则用 //包裹

[abc] :任意取一个值

[a-z] : 表示一个范围

[0-9] :

\d : 数字

\D : [^0-9]

\w : 英文数字和下划线

\W : 非英文数字下划线

\s : 空白字符

\S : 非空包字符

^ : 非

. : 任意字符

量词:

{n} : 正好出现n次

{n.m} : 最少n次,最多m次

{n,} : 最少n次,最多不限

+ : 最少出现1次

? : 最少0次,最多1次

*:可有可没有(0或0以上),贪婪匹配

// i : 忽略大小写

^放在开头 :行首

$ : 行尾
